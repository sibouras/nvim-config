# My nvim config

## Requirements

- System requirements for [nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter/tree/main?tab=readme-ov-file#requirements)
- Terminal emulator with [true colors](https://github.com/termstandard/colors#truecolor-support-in-output-devices) and [Nerd Font icons](https://www.nerdfonts.com/)
- [ripgrep](https://github.com/BurntSushi/ripgrep#installation)

## Installation

```bash
git clone https://github.com/sibouras/nvim-config.git
:Lazy restore
```
